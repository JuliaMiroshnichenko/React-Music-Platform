
## Fullstack Музыкальная платформа / Fullstack Music Platform

---

### Инструменты / Tools

* React ( Server side rendering ) 
* Next js
* Typescript (  Серверная часть )

---

### Инструкция по запуску:

1. npm run start:dev - запуск сервера

2. npm run dev - запуск клиента
